Platform Purpose / 平台目的

The gonggong platform is intended to become a single-point platform for the platform owner's work and life workflows.

弓公平台的方向，是成為平台主人工作與生活流程的單點平台。

The platform is not presented here as a public personal profile, public business website, or public record of private files.

This public page only gives cloud AI enough public-safe context to understand the platform direction at a high level.

Work-Life Separation / 工作與生活分離

The platform owner expects work data and life data to remain separated.

平台主人預期工作資料與生活資料應分開管理。

This page does not define technical storage, account structure, database schema, folder structure, NAS setup, or implementation design.

本頁不定義技術儲存方式、帳號結構、資料庫 schema、資料夾結構、NAS 設定或實作設計。

Public Context Purpose / 公開背景用途

Public pages exist to provide cloud AI with public-safe context for understanding platform direction, role boundaries, planning style, and non-authorization rules.

公開頁面的用途，是讓雲端 AI 能以 public-safe 的方式理解平台方向、角色邊界、規劃風格與不授權規則。

These pages should be treated as public reference-only context. They do not replace private formal governance records.

Cloud AI Role / 雲端 AI 角色

Cloud AI should use this public context mainly to assist thinking, planning, comparison, proposal drafting, and discussion.

雲端 AI 可使用本公開背景協助思考、規劃、比較、提案草稿與討論。

Cloud AI must not treat this page as execution authorization, repository authorization, or a complete governance source.

What This Page Does Not Include / 本頁不包含內容

  • Private files or private repository contents.
  • Complete private governance records.
  • Complete role startup pack contents.
  • Complete task orders or auditor reports.
  • Financial records, business-sensitive details, family information, or health information.
  • NAS paths, local private paths, credentials, tokens, passwords, API keys, implementation secrets, production details, or deployment details.

Unknown / Unavailable Rule / 未知與不可取得規則

If current repository state, local files, private decisions, or task history are not directly available, AI must not infer them from public pages.

AI should mark them as unknown or unavailable, or ask the user to provide the relevant context.

若目前 repository 狀態、本機檔案、私有裁示或任務歷史無法直接取得，AI 不得從公開頁推測。

AI 應標示 unknown / unavailable，或請使用者提供相關脈絡。
